我想做一个安卓项目，需求是扫描二维码解析文本。
页面布局：
1.页面上下布局，上半部分是相机取景框，下半部分是文本框。
2.文本框底部按钮：复制、撤销、恢复。文本框右上角按钮：清空（清空文本框）、删除（退格键，逐字删除）
3.取景框右上角：关闭/开启（关闭开启摄像头）
4.应用点开后仅当前页面，简单直接，没有二级页面
功能：
1.支持识别二维码后，将文本放入文本框
2.支持连续扫描二维码，将文本内容持续累加进入文本框中
3.撤销按钮：支持撤销单次二维码识别的文本内容
4.恢复按钮：支持恢复撤销的文本内容
5.复制按钮：支持将文本框中内容进行复制
6.识别到二维码后，取景框提示扫描成功动效，不要出现文字，以图形样式提醒扫描成功
